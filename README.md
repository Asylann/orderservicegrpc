# Order Service

## Description

#### This is my order service on grpc for working on orders managing for my Web_Sportwear_Shop

### Main specifics:
* With user and cart id and also transportation type and address, it builds an order with order_items and also creatAt and DeliveredAt time.
* It gets all items from cart and be connected with cart service also.
* It also resposenble for informing sellers about orders they have to deliver (maybe further i will also create messaging service for informing sellers and also users)!
